# NumSynth Comparison

This folder contains scripts and datasets for comparing PyGol+Z3 with NumSynth.

## Quick Start

Run full comparison:
```bash
cd numsynth_comparison
python run_full_comparison.py
```

## Files

- **`run_full_comparison.py`**: Main comparison script (PyGol+Z3 vs NumSynth on all 35 problems)
- **`convert_to_numsynth.py`**: Converts datasets to NumSynth-compatible format
- **`numsynth_datasets/`**: Converted datasets in NumSynth format
- **`numsynth-main/`**: NumSynth source code

## Results

Results saved to `comparison_results.txt` and `comparison_results.json`.
